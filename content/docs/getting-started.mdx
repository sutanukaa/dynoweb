---
title: Get Started
description: Install DynoWeb, activate tracking, and begin using the core analysis tools.
---

Install DynoWeb and click Get Started.

## Enable Tracking

Open Shopify Theme Editor, enable the DynoWeb app embed, save, and verify installation.

Visit the storefront yourself or wait for live traffic. The Dashboard banner confirms when the tracker is active and when the last event arrived.

## Session Replay

If you want recorded sessions, turn on Session Replay in Settings.

Replay is off by default and can be limited to frustrated sessions only.

## First Places To Check

Start with Dashboard for store-wide health, then use Heatmaps for page-level interaction, Journeys for drop-off paths, Conversions for attributed revenue, Cart Overview for product intent, and Replays for session-level inspection.

## Suggestions And Analysis

Open Suggestions and run Quick Store Check right away.

Run Traffic Analysis after enough real visitor data has been collected.

Preview supported suggestions — currently CSS suggestions are available and can be applied to a draft theme. More suggestion types are coming soon. Test in Shopify and publish only when ready.

## Measure Results

After a draft change is live, use Experiments to compare before-and-after performance.

Use DynoAgent for follow-up analysis and, on eligible plans, approved store actions.
